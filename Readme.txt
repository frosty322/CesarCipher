Курсовая работа выполнена при помощи стека технологий .net, а именно на базе ASP.net MVC. 
    Реализация включает в себя использование библиотеки работы с документами Microsoft Office - NPoi,
    синтаксиса разметки для внедрения в веб-страницы HTML серверного кода - Razor,
    фреймворка из трех языков HTML/CSS/JS для оформления веб-страницы - Bootstrap 4.
    Программа выполняет функцию кодировки и декодировки текста, заданного как из рабочего окна, так и из файла, декодировка проводится в двух режимах:
    с ключем и без ключа. Поиск нужного ключа реализован на основе частотного анализа. Есть возможность скачивания файла с текстом в формате Docx, проведены UnitTest
